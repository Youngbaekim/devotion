매일묵상 앱 바로가기 설치

GitHub Repository에 아래 파일 3개를 추가하면
스마트폰 바탕화면에 앱처럼 설치됩니다.

📁 추가할 파일 목록

1
manifest.json — 앱 이름·색상·아이콘 정의
2
sw.js — 서비스워커 (오프라인 지원)
3
icon-192.png / icon-512.png — 앱 아이콘 이미지

📝 index.html <head>에 추가할 코드

기존 <head> 태그 안에 아래 4줄을 복사·붙여넣기 하세요.

<link rel="manifest" href="/devotion/manifest.json"> <meta name="apple-mobile-web-app-capable" content="yes"> <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent"> <meta name="apple-mobile-web-app-title" content="매일묵상"> <meta name="theme-color" content="#2d5a8e"> <link rel="apple-touch-icon" href="/devotion/icon-192.png">

⚙️ </body> 직전에 추가할 코드

서비스워커 등록 스크립트입니다.

<script> if ('serviceWorker' in navigator) { navigator.serviceWorker.register('/devotion/sw.js'); } </script>

📱 설치 방법

파일 업로드 후 아래 방법으로 설치하세요.

🍎
iPhone (Safari)
Safari에서 사이트 열기 → 하단 공유 버튼 → 홈 화면에 추가
▲
Android (Chrome)
Chrome에서 사이트 열기 → 우상단 메뉴(⋮) → 홈 화면에 추가
설치 완료 시 주소 표시줄 없는 전체화면 앱으로 실행됩니다 🙏
